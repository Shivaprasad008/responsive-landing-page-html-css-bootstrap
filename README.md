## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/Shivaprasad008/responsive-landing-page-html-css-bootstrap.git
cd responsive-landing-page-html-css-bootstrap
```

### 2. Install dependencies

```bash
npm install
```

> This uses the `package.json` and `package-lock.json` that already exist in the repo [page:70].

### 3. Start the development server

```bash
npm run dev
```

This will start the Vite dev server. Open the URL shown in the terminal, typically:

```text
http://localhost:5173
```

to view the responsive landing page in your browser [web:76].

### 4. Build for production (optional)

```bash
npm run build
```

### 5. Preview the production build (optional)

```bash
npm run preview
```

These commands are standard for a Vite + TypeScript project with `vite.config.ts`, `tsconfig*.json`, and `src/` present in your repo [page:70][web:76].
